# Cleyri Solano García — Portafolio

Portafolio personal desarrollado con HTML, CSS y JavaScript vanilla.  
Disponible en: **[cleyri-solano.github.io](https://cleyri-solano.github.io)**

---

## Sobre el proyecto

Este portafolio fue construido desde cero sin frameworks de UI, con foco en diseño limpio, código semántico y rendimiento. Incluye animaciones con Intersection Observer, scroll spy en la navegación, menú responsive y barra de progreso de scroll.

## Secciones

- **Hero** — presentación con links directos a GitHub, LinkedIn y descarga de CV
- **Sobre mí** — perfil profesional y enfoque técnico
- **Proyectos** — 4 proyectos destacados con capturas reales y links a repositorios
- **Stack Tecnológico** — tecnologías organizadas por categoría
- **Educación** — formación académica y bootcamp
- **Soft Skills** — habilidades blandas
- **Contacto** — links directos para conectar

## Stack del portafolio

- HTML5 semántico
- CSS3 (variables, grid, flexbox, animaciones)
- JavaScript vanilla (Intersection Observer, scroll spy)
- [Devicon](https://devicon.dev/) para íconos de tecnologías
- Google Fonts — Playfair Display + Plus Jakarta Sans
- Desplegado en GitHub Pages
